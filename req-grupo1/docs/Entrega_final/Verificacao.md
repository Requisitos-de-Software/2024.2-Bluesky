## Introdução

Durante todo o projeto foram construídas listas de verificação referentes aos artefatos produzidos em cada entrega, além disso, o projeto contou com uma voltada para verificação e validação. Com isso em mente, este artefato é uma síntese das verificações realizadas pelo grupo 01, tanto de seus próprios artefatos, quanto dos artefatos do grupo 2 em comum.

## Objetivo

O objetivo deste artefato é auxiliar a consulta e a gestão das verificações realizadas, propiciando uma visão geral dos artefatos inspecionados e indicando os realizadores das inspeções e os construtores das listas de verificação.
